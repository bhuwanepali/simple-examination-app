
# Laravel 10 Simple Examination Project



## Run Locally

Clone the project

```bash
  git clone https://github.com/bhuwanepali/simple-examination-app.git
```

Go to the project directory

```bash
  cd simple-examination-app
```

Install dependencies

```bash
  npm install
```
Migration

```bash
  php artisan migrate
```

Start the server

```bash
  php artisan serve
```


```bash
  npm run dev
```
